Preguntas Frecuentes(FAQs)

	

-¿Que hacer cuando ocurre un error de que no ha podido copiar el archivo a la carpeta correspondiente?
	
	Se recomienda ver el error y revisar en que  ruta ha fallado y comprobar si puede ser por algún permiso que no este bien habilitado.

-¿Que hacer cuando al instalar el programa no encuentra las carpetas propias del postgres?
	
	Esto puede ser debido a que al borrar y volver a instalar el postgres se hayan cacheado, para ello lo que se recomienda  es realizar un borrado del postgres, posteriormente reiniciar el ordenador, comprobar que se han borrado los archivos y volver a instalarlos*.

-¿Que hacer si cuando intento ingresar la contraseña por primera vez me dice que ingrese una?
	
	Esto puede ocurrir debido a que se ha quedado cacheado la base de datos. PAra solucionarlo tendras que deistalarla y luego reiniciar, una  vez reiniciado volver a hacer la instalacion*.
			
			
Se recomienda instalar el programa pgadmin3 para instalar los comando sql y crear la base de datos en caso de que se quiera generar .
